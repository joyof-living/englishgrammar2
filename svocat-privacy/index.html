에스보냥 (SVOCAt) 개인정보처리방침

최종 수정일: 2026-04-26 · 버전: 2.1.2

요약

본 확장 프로그램은 사용자의 개인정보를 수집하지 않으며, 개발자 소유의 서버로 전송하지 않습니다.

모든 설정은 사용자의 브라우저 로컬 저장소에만 보관됩니다.

분석 요청은 사용자가 선택한 AI 제공사로 직접 전송됩니다.

1. 사용자 기기에 저장되는 정보

본 확장 프로그램은 어떤 사용자 정보도 수집하거나 전송받지 않습니다. 다음 항목은 사용자 본인의 브라우저 로컬 저장소(chrome.storage.local)에만 저장되며, 개발자 서버나 제3의 장소로 전송되지 않습니다.

항목설명저장 위치
API 키Google Gemini, Groq, Vertex AI 등 (사용자가 직접 발급·입력)chrome.storage.local
모델 선택분석에 사용할 AI 모델chrome.storage.local
분석 히스토리최근 분석 문장/결과 (최대 10건, 로컬 전용)chrome.storage.local

이메일, 이름, 위치 등 개인 식별 정보는 일체 다루지 않습니다.

2. 정보의 사용

  • API 키: 사용자가 텍스트 분석을 요청할 때, 사용자가 선택한 AI 제공사의 API 엔드포인트로만 전송됩니다.
  • 선택한 텍스트: 동일한 AI 제공사로 전송되어 문법 분석과 번역에 사용됩니다.
  • 분석 히스토리: 사이드패널의 "최근 분석" 기능을 위해 로컬에만 보관됩니다.
  • 개발자는 이 정보에 접근할 수 없습니다.

3. 외부 서비스 (제3자 제공)

선택한 텍스트는 사용자가 선택한 다음 AI 제공사 중 하나로 전송됩니다.

제공사엔드포인트개인정보처리방침
Google (Gemini) generativelanguage.googleapis.com 링크
Google (Vertex AI) aiplatform.googleapis.com 링크
Groq api.groq.com 링크

각 제공사의 약관이 해당 전송에 적용됩니다. 본 확장 프로그램은 단순 중계자이며 전송 내용을 열람·저장·가공하지 않습니다.

4. 쿠키 및 추적

  • 쿠키를 사용하지 않습니다.
  • 사용자 활동을 추적하지 않습니다 (클릭, 스크롤, 사용 텔레메트리 없음).
  • Google Analytics, Mixpanel 등 분석 도구를 포함하지 않습니다.
  • 광고 또는 마케팅 목적의 정보 수집을 하지 않습니다.

5. 데이터 저장 및 삭제

  • 모든 데이터는 브라우저 로컬 저장소에만 저장됩니다.
  • 확장 프로그램 제거 시 관련 데이터가 함께 삭제됩니다.
  • 개별 삭제: 옵션 페이지의 "저장된 키 삭제" 버튼을 이용하세요.
  • 완전한 보안을 위해, 삭제 후 잔존 데이터가 우려되면 발급처(Groq, Google)에서 API 키를 직접 폐기(rotate)하시기 바랍니다.

6. 보안

  • API 키는 Chrome의 자체 암호화 정책에 따라 저장됩니다.
  • AI 제공사와의 통신은 HTTPS로만 이루어집니다.
  • 기기 도난 위험 방지를 위해 OS 수준 암호화(BitLocker, FileVault)를 권장합니다.

7. 어린이의 개인정보 보호

본 확장 프로그램은 만 13세 미만 어린이를 대상으로 하지 않습니다.

8. 정책 변경

본 방침은 향후 수정될 수 있으며, 중요 변경사항은 확장 프로그램 업데이트 릴리스 노트에 공지됩니다. 변경 시 본 페이지의 "최종 수정일"이 갱신됩니다.

Privacy Policy for 에스보냥 (SVOCAt)

Last Updated: 2026-04-26 · Version: 2.1.2

Summary

This extension collects no user data and does not transmit any data to developer-owned servers.

All settings are stored only in the user's local browser storage.

Analysis requests are sent directly to the AI provider chosen by the user.

1. Data Stored on Your Device

This extension collects no user data. The following items are stored only in your browser's local storage (chrome.storage.local) and are never transmitted to the developer or any third-party server:

  • API Key — for Google Gemini, Groq, or Vertex AI (issued and entered by the user)
  • Model Selection — chosen AI model for analysis
  • Analysis History — up to 10 recent analyses (local only)

We do not handle emails, names, locations, or any other personally identifiable information.

2. How Information Is Used

  • API key and selected text are sent only to the AI provider chosen by the user.
  • Analysis history is stored locally only, for the "Recent Analyses" feature.
  • The developer has no access to any of this data.

3. Third-Party Services

When requesting analysis, the selected text is sent to one of: Google (Gemini / Vertex AI) or Groq. Each provider's privacy policy applies. This extension is merely a relay; it does not read, store, or process the text content on its own servers (it has none).

ProviderEndpointPrivacy Policy
Google (Gemini) generativelanguage.googleapis.com Link
Google (Vertex AI) aiplatform.googleapis.com Link
Groq api.groq.com Link

4. Cookies, Tracking, Analytics

None. No cookies, no tracking, no analytics tools (Google Analytics, Mixpanel, etc.), no advertising.

5. Data Deletion

  • Uninstalling the extension removes all data.
  • Use the "Delete saved key" button in options to remove the API key.
  • For complete security, rotate your API key at the provider (Groq, Google).

6. Security

  • HTTPS-only communication.
  • API keys stored per Chrome's own encryption policy.

7. Children's Privacy

This extension is not directed at children under 13.

8. Changes

This policy may be updated. Check the "Last Updated" date above.
