에스보냥에 오신 걸 환영합니다

영어 문장을 드래그하면 SVOCA 구조로 분석하고 한국어로 번역합니다.
시작하려면 AI API 키를 발급받아 아래에 붙여넣어 주세요.

1. Google Gemini API 키 발급

Google Gemini

권장

한국어 품질 최상 · 무료 할당량 제공 · 약 2분 소요

  1. Google 계정으로 로그인
  2. "Get API key" → "Create API key in new project" 선택
  3. 생성된 키(AIza...)를 복사해서 아래에 붙여넣기
Gemini 키 받기 →
과금 주의 — Gemini API는 무료 할당량을 제공하지만 초과 시 Google Cloud 계정에 과금이 발생할 수 있습니다. 사용량과 결제 설정은 Google Cloud Console에서 확인·제한 설정하실 수 있습니다.

고급 사용자는 Google Vertex AI 또는 Groq 키(gsk_...)도 붙여넣어 사용할 수 있습니다. 확장이 자동으로 감지합니다.

키는 기기에만 저장되며 외부로 전송되지 않습니다 (분석 요청 시 Google API에만 직접 전송).

준비 완료

이제 아무 웹페이지에서 영어 문장을 드래그하면

"문법 분석" 버튼이 나타납니다.

버튼을 클릭하면 사이드패널이 자동으로 열리면서
SVOCA 구조 분석과 한국어 번역을 보여줍니다.

설정은 언제든 확장 아이콘 클릭 후 기어(⚙) 버튼으로 변경할 수 있습니다.
